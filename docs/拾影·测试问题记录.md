## 修复记录

### 2026-05-26

**#2 AI测试连接崩溃**
- 文件：`ui/main_window.py`
- 原因：`_test_ai_connection` 未处理 httpx 异常，连接失败时程序崩溃
- 修复：添加 try-catch 包裹异步调用，异常时返回 False 并提示用户，finally 中关闭 agent

**#3 AI配置默认显示**
- 文件：`ui/sidebar.py`
- 原因：AI 模型配置 GroupBox 默认可见，容易误触发
- 修复：默认隐藏 `model_group.hide()`，监听模式切换信号，选择 AI 模式时 `model_group.setVisible(True)`

**#4 图片加载卡在11%**
- 文件：`core/image_loader.py`
- 原因：批次触发逻辑仅在 `_loaded_count % THUMBNAIL_LOAD_BATCH == 0` 时推进，加载失败的图片不计数，导致批次卡住
- 修复：改用 `_processed_count` 统计已处理数（成功+失败），错误也触发 `_check_batch_complete()` 推进批次
